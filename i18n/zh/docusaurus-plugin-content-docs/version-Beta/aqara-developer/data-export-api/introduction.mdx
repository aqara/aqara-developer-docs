# 简介

## 概述

Aqara 数据转出 API 提供了一套丰富的接口，支持用户高效、安全地访问和管理 Aqara Studio 内的设备、空间与自动化相关的数据。通过这些 API，用户可以实现空间和设备的查询、设备的远程控制、Trait 功能点的读取与下发，以及自动化规则的创建、查询、更新和删除等操作。无论是进行设备管理还是构建自动化场景，数据转出 API 都能为开发者提供基础能力，助力快速集成和扩展 Aqara 平台的数据与服务。
