# 概览

阅读本文档，您可以了解 Aqara 为您进一步开发提供的资源。

## 开发者资源

### 开始前请先选择接入路径

Aqara 当前提供的 API 分为两类，它们面向的环境和前置条件不同。建议您先确认要接入的是 **Cloud API** 还是 **Studio API**，再阅读对应文档。

| API 类型 | 适用场景 | 开始前需要准备 |
| --- | --- | --- |
| Cloud API | 接入 Aqara 云端开放平台能力 | 访问 [Aqara 开发者平台](https://developer.aqara.com/)，创建应用，获取 `Appid`、`Keyid`，并完成授权与签名准备。 |
| Studio 本地 API | 接入本地 Aqara Studio 内的数据与控制能力 | 先访问并准备 [Aqara Studio](../aqara-studio/overview/introduction.mdx) 环境，获取服务地址、Token 以及相关项目或设备数据。 |
