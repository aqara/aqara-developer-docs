# 概览

阅读本文档，您可以了解 Aqara 为您进一步开发提供的资源。

## 开发者资源

### Aqara 开发者平台

在开始调用接口或使用相关开发资源之前，建议先访问 [Aqara 开发者平台](https://developer.aqara.com/)，完成应用创建、凭证申请、授权配置以及基础接入准备。

### API 文档

本栏目会逐步提供多组 API 文档，用于覆盖不同产品能力与接入场景。当前已经提供的 API 文档可用于了解接口范围、请求方式、认证要求、参数结构以及响应格式。

### SDK 与开发工具

除 API 文档外，这里也将持续补充 SDK、调试工具、集成示例以及其他开发辅助资源，帮助您更高效地完成接入、验证与联调。

### 在线调试与验证

对于支持在线调试的 API 文档，您可以直接在文档页面中验证请求参数、请求头、签名和服务地址配置，缩短联调与排查问题的时间。
