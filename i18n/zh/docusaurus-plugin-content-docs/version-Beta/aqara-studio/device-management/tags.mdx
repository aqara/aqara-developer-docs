# 添加标签

为便于灵活组织与精准识别智能家居系统中的各类设备与功能点，Aqara Studio 支持您为协议、设备或点添加多种类型的标签，提升设备管理的结构与效率。

## 标签类别

Aqara Studio 将标签分为 [标记标签](#标记标签) 和 [值标签](#值标签) 两类。

- **标记标签**：用于标识和分类协议、设备和功能点，不可赋值。
- **值标签**：支持为其设置具体取值，以便描述协议、设备或功能点的特定属性。

## 添加流程

下文介绍如何在 Aqara Studio 中为**协议**、**设备**或**点**添加标签，涵盖标记标签和值标签的具体操作步骤。

1. 进入**设备管理**页面，在左侧设备树中找到待添加标签的**协议**、**设备**或**点**。
2. 点击该设备或点，查看其 **设备详情**选项卡。

<img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Device-Tags.png" alt="找到标签" style={{ width: '80%' }} />   


### 标记标签

1. 在**标记标签**区域，点击**添加**按钮。
2. 选择或输入标记标签后，点击**确定**按钮，所添加的标签会显示在**标记标签**区域。

<img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Device-AddMarkTag.png" alt="添加标记标签" style={{ width: '80%' }} />   

### 值标签

1. 在**值标签**区域，点击**添加**按钮。
2. 选择值标签，并赋值。
3. 设置完成后，点击**确定**，新增的值标签即显示在该区域。

<img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Device-AddValueTag.png" alt="添加值标签" style={{ width: '80%' }} />
