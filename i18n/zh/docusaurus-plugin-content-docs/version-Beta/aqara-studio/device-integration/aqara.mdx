# Aqara Hub 接入

:::tip
本文档仅适用于 **网关融合形态**的 Aqara Studio。
:::

### 查看 Aqara Hub 设备

在 **网关融合形态** 的 Aqara Studio 中，已默认集成所搭载的 Aqara Hub（如智慧边端 M300），无需手动添加。您只需在 **设备管理** 页面展开设备树下的 `Aqara Hub` 节点，即可查看对应设备。

### 如何添加其他类型的 Aqara 设备

如需接入不同协议的 Aqara 设备，请参考以下文档，按设备协议类型进行集成操作：
- [接入 Zigbee 设备](./zigbee.mdx)
- [接入 Matter 设备](./matter.mdx)
