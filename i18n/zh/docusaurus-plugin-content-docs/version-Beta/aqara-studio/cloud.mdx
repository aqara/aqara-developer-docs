# 云端连接

## 功能简介

云端连接（Edge2Cloud）可以将本地 Studio 绑定至 [Aqara Builder](https://builder.aqara.com) 云平台，以便用户在任何地方都可以通过 Aqara Builder 远程访问本地 Studio ，不再受到本地访问的限制。

主要功能包括：
- 为云平台具备权限的用户提供 [远程安全访问](./remote-access.mdx) 服务。
- 可通过云平台远程监控等云端运营功能。
- 为本地 Studio 提供更多可扩展的增值服务。
- 提供本地 Studio 和云端平台之间的数据交互功能。
- 用户远程访问在所有维护的 Studio 版本上均免费使用。   

更多功能尽情期待!

## Aqara Builder 简介

作为 Aqara 的云中心，Aqara Builder 连接了本地的 Aqara Studio 和基于云的创意工作流程。Aqara Builder 支持多个现场站点集中管理 Aqara Studio 设备，并实现一键远程连接，实现无缝云端运营。即使没有 Studio 设备，用户也可以通过“试用在线演示”功能体验完整的 Aqara Studio 创作过程。

## 使用流程

1. 从左侧侧边栏访问**云端连接**页面。
2. 在该页面登录 Aqara Builder 账号。
3. 将 Aqara Studio 与您的 Aqara Workspace 绑定。
