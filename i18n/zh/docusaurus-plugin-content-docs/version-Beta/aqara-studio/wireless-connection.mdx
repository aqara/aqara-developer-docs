# 无线连接

:::tip
本文档仅适用于 **网关融合形态**的 Aqara Studio。
:::

本文说明如何连接无线网络。

## 流程 

1. 在左侧侧边栏点击进入**设置 > 本机设置**页面。
2. 找到 **网络设置**，输入待接入 Wi-Fi 的 名称和密码，单击 **保存设置** 按钮。
3. 拔掉网线，即可使 Aqara Studio 使用无限网络。
