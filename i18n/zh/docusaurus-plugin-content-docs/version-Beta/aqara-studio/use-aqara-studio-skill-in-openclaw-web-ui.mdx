# 在 OpenClaw Web UI 中使用 Aqara Studio Skill

## 功能概述

Aqara 提供了用于集成 Aqara Studio 能力的 OpenClaw skill。 OpenClaw Web UI 中实现：

- 列出设备、统计设备数量、读取状态、控制设备；
- 列出空间、创建或重命名空间、在空间之间移动设备；
- 创建自动化、查看自动化、更新自动化配置、启用或禁用自动化、删除自动化。

整个过程无需编写代码，仅需简单配置必要的信息，即可实现对 Aqara 设备的全局管控，为您的智能空间管理带来极大便利。

## 安装 Skill

1. 为您的 OpenClaw 安装 [Aqara OpenAPI Local](https://clawhub.ai/aqara/aqara-open-api-local) Skill。例如，您可以直接在输入框中输入 `安装 Aqara OpenAPI Local Skill`。Assistant 即会开始安装 Skill。

2. 安装完成后，在左侧边栏点击 **Skills**，若在 **WORKSPACE SKILLS** 列表中能看到 `aqara-open-api-local`，则说明安装成功。

    ![确认 Aqara API 是否已安装](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/OpenClaw-Skills.png)

## 配置 Skill

1. 安装完成后，Assistant 会要求您提供 `AQARA_ENDPOINT_URL` 和 `AQARA_OPEN_API_TOKEN`，用作鉴权。

    :::tip
    - `AQARA_ENDPOINT_URL` 为 Aqara Studio 的 [HTTP 请求 URL](./api-references/data-export-api-v1.mdx#http-请求)，通常以 `/open/api/v1` 结尾
    - `AQARA_OPEN_API_TOKEN` 为 Aqara Studio 用于鉴权的 [token](./developer-guide.mdx#获取必要信息)。
    :::

    ![输入 url 和 token](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/openclaw-url-and-token.png)

    提交上述信息后，OpenClaw 会自动完成所需配置。
2. 服务配置成功后，您即可通过 OpenClaw 实时查看和管理 Aqara Studio 已接入的所有设备，实现高效的设备监控与便捷控制。

    例如，你可以提问：Aqara Studio 内集成了多少设备？
    
    ![提问集成设备数量](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/openclaw-query-device-list.png)


## 常见问题

### 如何查看已传入的 AQARA_ENDPOINT_URL 和 AQARA_OPEN_API_TOKEN？

1. 在 **Chat** 会话中要求 Assistant 输出 openclaw.json 文件。

    ```txt
    Output the openclaw.json file
    ```

2. Assistant 会将 openclaw.json 文件的内容展示在对话框中，您可以在其中查找到已配置的 `AQARA_ENDPOINT_URL` 和 `AQARA_OPEN_API_TOKEN`。

    ![查看 URL 和 Token](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/openclaw-check-info.png)
