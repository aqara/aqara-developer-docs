# Wireless Connection

:::tip
This document is only applicable to the **hub-integrated model** of Aqara Studio.
:::

This article explains how to connect to a wireless network.

## Process

1. In the left sidebar, go to **Settings > Local Settings**.
2. Find **Network Configuration**, enter the Wi-Fi name and password you want to connect to, and click the **Save settings** button.
3. Unplug the network cable to allow Aqara Studio to use the wireless network.
