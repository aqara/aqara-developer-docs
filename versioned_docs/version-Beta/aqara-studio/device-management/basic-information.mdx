# View and Edit Basic Information

This article explains how to view and edit the basic information of a device, such as its name.

1. Go to the **Devices** page. In the device tree on the left, find the device folder you want to view.
2. Double-click the device folder to open the device **Essential** tab. Here, you can view and edit the following basic information:

    | Field        | Editable | Description |
    |--------------|----------|-------------|
    | Device Name  | ✓        | You can rename the device to help with identification and management. |
    | Device ID    | ✗        | The ID assigned to the device by Aqara Studio. |
    | Device Status| ✗        | The device's online status. |
    | Device Type  | ✗        | A collection of the function point types of the device. |
    | Space        | ✓        | It Shows and allows you to change the space where the device is located. This helps you manage the mapping between devices and physical spaces. |
    | Import Time  | ✗        | The time when the device was imported into Aqara Studio. |
