# Aqara Hub

:::tip
This document is only applicable to the **hub-integrated model** of Aqara Studio.
:::

This guide explains how to connect Aqara Hub to Aqara Studio.

### Locate Aqara Hub

In the **hub-integrated model** of Aqara Studio, the built-in Aqara Hub (such as the Edge Hub M300) is integrated by default, so no manual addition is required. 

You only need to expand the `Aqara Hub` node in the device tree on the **Devices** page to view the related devices.

### How to Add Other Types of Aqara Devices

To integrate Aqara devices using different protocols, please refer to the following documents and follow the integration process according to the device’s protocol type:
- [Zigbee](./zigbee.mdx)
- [Matter](./matter.mdx)
