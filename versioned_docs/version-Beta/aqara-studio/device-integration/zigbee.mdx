# Zigbee

:::tip
This document is only applicable to the **hub-integrated model** of Aqara Studio.
:::

This document explains how to connect Zigbee devices to Aqara Studio.

:::tip
Before connecting a Zigbee device, it is recommended to check the [Compatible Aqara Devices](./../overview/compatible-aqara-devices.mdx) list to ensure it is supported.
:::

## Process

1. In the left sidebar, click to enter the **Devices** page. Select `Zigbee` to open the management page.

    ![Select Zigbee](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/Devices-Zigbee.png)

2. Go to the `Add Device` tab. In the **Select gateway** dropdown, select the gateway to which you want to bind sub-devices. Under **Opening method**, choose the pairing mode for the gateway (supports **Tealtime** or **Timing**).

    Click the **Enable Network Access** button. The selected gateway will enter pairing mode. Now put the sub-device into pairing mode to complete the binding.

    ![Configuration](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/Zigbee-Configuration.png)

3. Once the sub-device is successfully bound, it will appear in the **Device added** list.

## Next Steps

Now, you can [view details and control the device](./../device-management/basic-information.mdx), or [use the device in automation](./../automation.mdx).

## FAQ

### How do I connect Aqara devices that support dual-protocol or multi-protocol by Zigbee?

If your Aqara device supports Zigbee as well as other protocols, please follow these steps to connect via Zigbee:

- If the device is already in Zigbee mode, you can connect following the process described in this document.
- If the device uses another protocol, first switch the device to Zigbee mode on the Aqara Home App. After switching, follow this document’s process to connect.
