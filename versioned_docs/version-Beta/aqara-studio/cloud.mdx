# Cloud Connect

## Feature Overview

Cloud Connect (Edge2Cloud) allows you to bind your local Studio to the [Aqara Builder](https://builder.aqara.com) cloud platform, so that you can remotely access their local Studio from anywhere via Aqara Builder, free from local network restrictions.

Key features include:
- Provides [secure remote access](./remote-access.mdx) services for cloud platform users with the necessary permissions.
- Enables remote monitoring and other cloud-based operational features through the cloud platform.
- Offers more extensible value-added services for local Studio.
- Facilitates data interaction between the local Studio and the cloud platform.
- Remote access for users is free of charge across all maintained Studio versions.

More features are coming soon!

## About Aqara Builder

Serving as the definitive cloud hub, [Aqara Builder](https://builder.aqara.com) bridges local Aqara Studio and cloud-based creative workflows. Aqara Builder enables centralized management of Aqara Studio devices across multiple on-site locations and one-click remote connection to unlock seamless cloud-based operations. 

Even without Studio, users can experience the complete Aqara Studio creative process through the 'Try Online Demo' feature.

## How to Use

1. Go to the **Cloud Connect** page from the left sidebar.
2. Log in with your Aqara Builder account on this page.
3. Bind Aqara Studio with your Aqara Workspace.
