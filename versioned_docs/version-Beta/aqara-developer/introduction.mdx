# Overview

Learn about the resources Aqara provides for your further development.

## Developer Resources

### Aqara Developer Platform

Before using APIs or related developer resources, start in the [Aqara Developer Platform](https://developer.aqara.com/) to create your app, request credentials, configure authorization, and prepare the basic integration setup.
